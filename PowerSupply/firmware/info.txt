esp32_start.bin - файл для первичной загрузки в esp32. 

esp32_ps_ssd1306_210723.bin обновлять при использовании дисплея с контроллером SSD1306
esp32_ps_st7567s_210723.bin обновлять при использовании дисплея с контроллером st7567s

Последующие обновления через механизм OTA в WEB интерфейсе.
https://github.com/pr200sd/presp32/wiki/4.11-Firmware-Upgrade-(OTA)

После установления связи через Web интерфейс необходимо выбрать во вкладке DIO режим Power Supple, сохранить настройку и перезагрузить контроллер.

Для снятия DEMO режима прислать ID на указанную почту.
