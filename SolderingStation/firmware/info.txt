esp32_start.bin - файл для первичной загрузки в esp32. 
 
паяльная станция, дисплей с контроллером st7567s
esp32_ss_st7567s_v1_190424 - схема с внешним ТТР, управление по "+"
esp32_ss_st7567s_v2_190424 - схема оптрон+триак, управление по "-"

Последующие обновления через механизм OTA в WEB интерфейсе.
https://github.com/pr200sd/presp32/wiki/4.11-Firmware-Upgrade-(OTA)

После установления связи через Web интерфейс необходимо выбрать во вкладке DIO режим Soldering Station, сохранить настройку и перезагрузить контроллер.

Для снятия DEMO режима прислать ID на указанную почту.
